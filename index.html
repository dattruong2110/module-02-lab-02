Frontend là gì

Frontend, Dec 1, 2020

Lập trình Frontend là...

Lập trình Front End là việc sử dụng các ngôn ngữ HTML, CSS hay ngôn ngữ lập trình Javascript để các lập trình viên thiết kế ra các giao diện ứng dụng hoặc trang web cho người dùng.

Backend là gì?

Backend, Dec 1, 2020

Lập trình Backend là...

Lập trình Backend có nghĩa là làm việc trên phần mềm phía máy chủ nhằm xử lý và cung cấp dữ liệu cho phía front-end. Nó bao gồm: cơ sở dữ liệu, logic backend, giao diện lập trình ứng dụng (API), kiến trúc và các chức năng phía máy chủ khác mà người dùng không thể nhìn thấy.
